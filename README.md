# LectureGenie

LectureGenie is a software program designed to transform your experience with recorded lectures. It utilizes advanced technologies to create a comprehensive and personalized learning environment:

* **Input any Lecture Video:** LectureGenie can process lectures with or without audio, making it adaptable to various recording formats. 
* **Generate Quiz Questions:** Test your understanding by creating quizzes directly from the video content. 
* **Summarize Key Points:**  LectureGenie automatically generates concise notes, highlighting the lecture's essential points.
* **Create Flashcards:** Solidify your knowledge by generating flashcards based on the lecture content.

**Technology Stack:**

* Python
* AWS
* Large Language Models (LLMs)
* APIs
* Computer Vision
* NumPy  
* Streamlit
